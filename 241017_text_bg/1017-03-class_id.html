CSS的選擇器

CSS有非常多選擇器，幫我們在撰寫樣式時，選擇到我們要的目標

Class 選擇器

特性:具有重複使用的特性，是使用率最高的選擇器

超連結

特性2:

抽離相同的屬性寫成 class 重複使用，再對不一樣的屬性定義 class

** class 是可以一直增加的 例如 box bg-color1 ....




id 選擇器

一份文件裡面，只允許一個獨一無二的id，就像身分證不能重複

對元素(標籤)設定 id，可以用城市快速取得它。

網頁區塊名稱會用 id 來命名

對元素(標籤)設定 id
